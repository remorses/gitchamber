# gitchamber

CLI tool that fetches source code for packages into `node_modules/.gitchamber/` to give coding agents deeper context. Fork of [opensrc](https://github.com/vercel-labs/opensrc).

Always run `gitchamber --help` before using this CLI. The help output contains all commands, options, defaults, and usage examples.

For subcommand details: `gitchamber <command> --help`
